Installation

Installation is straightforward with only one file needing to have a change made to it to get Weather Display Live working in evaluation mode. Step by step instructions are listed below.

To install Weather Display Live on your web server -

  1. Open the file index.html from the 'uploads' folder (which can be found in the folder that you installed Weather Display Live on your computer) into in an html editor, Notepad, or a similar text editor.

    Locate the line which says "Enter your details here!!"

    Change the line beneath it which says -

                 var pathToWdlconfig = "http://www.yourwebsite.com/wdlconfig.xml";
                
    so that "http://www.yourwebsite.com/wdlconfig.xml" is the path to your wdlconfig.xml file on your website.

  2. Copy the contents of the 'uploads' folder into the same folder on the web server that the clientraw.txt and clientrawextra.txt files get FTP'd into by Weather Display. If you haven't configured Weather Display to do this then go to the FAQ section on the forum for the instructions.

  3. That's it!

Open up a web browser and navigate to the index.html file. You should see your live weather data with an Evaluation Notice displayed on top of it. Note you can give the index.html file any name you wish.
