Upgrading to New Releases

New versions of Weather Display Live will be given a new name for the swf file for two reasons: to avoid problems with browsers caching old versions and to avoid confusion over which version is being used.

If you are applying a new version of Weather Display Live all you need to do is rename the reference to it in the web page that hosts the swf file so that it now refences the new swf. With reference to the section on installation change the line that says -

var versionOfWdl = "wdlv4_15.swf";

to the new version so that it now reads

var versionOfWdl = "wdlv6_03.swf";

Upload the index.html file again to your website and refresh your browser to see the new version.

If you are using a configuration file for WDL prior to v6 then please note that many instruments will need re-positioning. This is because of changes that had to be made to WDL to be compatible with the WDL Configurator.

If you use the WDL Configurator to manage your configuration files then functionality in the new release will be updated to work with the new WDL swf file. If you use an old version of either the WDL swf or the WDL Configurator then there may be compatibility issues. They are designed to work together with the same version.
